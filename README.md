# LabEng - Fullstack

## Note 🗒️

This app controls concrete ruptures for civil engineering laboratory.

## How to use 🚀

- Clone the project
- PostgreSQL docker

```sh
git clone https://github.com/jilvanx/lab-eng.git
```

- Install the dependencies and run.

```sh
$ cd lab-eng
$ yarn install
$ yarn dev
```

Or

```sh
$ cd lab-eng
$ npm install
$ npm run dev
```
